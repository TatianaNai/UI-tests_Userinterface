**Описание**

Этот проект содержит автоматизированные UI-тесты для сайта Userinterface https://userinyerface.com

**Инструкция для запуска**

1. Клонировать данные репозитория:  
git clone  https://github.com/TatianaNai/UI-tests_Userinterface.git  
2. Запустить проект:  
запустить файл Userinterface.sln или использовать команду dotnet test        

**Реализованные тесты**

Тесты автоматизируют переходы между страницами/формами и проверяют их корректное открытие/закрытие. В частности:  
•	Загрузка главной страницы  
•	Переход с главной страницы на страницу с формами  
•	Переход с одной формы на другую на странице с формами  
•	Скрытие формы помощи  
•	Скрытие формы cookies  
•	Проверка таймера на странице с формами  

**Технологии**

Автотесты написаны на C# с использованием NUnit, применены различные библиотеки и фреймворки. При написании использован фреймворк Aquality Selenium.
